# CSE 221 Operating systems project UCSD

OS performance benchmarking project for CSE 221 - Graduate operating systems course at UCSD

---

# Instructions


## Building

```
cd cpu
make
```
## Running tests
```
./run <Number of Iterations>
```
Example
```
./run 100000
```

## Clean

```
make clean
```
